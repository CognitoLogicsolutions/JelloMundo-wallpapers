1. **Clone the repo:**
   ```bash
   git clone https://github.com/CognitoLogic/JelloMundo-Wallpapers.git
   ```
2. **Browse the Vault:** Navigate to `wallpapers/raw-ai/` for clean outputs or `wallpapers/human-remix/` for the "Snapchat Twist" versions.
3. **Set & Forget:** Right-click any image and select **"Set as Desktop Background."**
